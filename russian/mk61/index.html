Программируемый микрокалькулятор МК-61

Описание

Здесь расположены документы по Программируемому микрокалькулятору МК-61. Все указанные документы верны и для МК-52.

Документация

Скан официальной документации: elektronika MK-61.pdf.

Дополнительные возможности

В связи с ограничениями памяти часть микрокода калькулятора не содержит ограничения. В результате ПМК может выполнять не только официальные команды, но также дополнительно выполнять действия, не предусмотренные документацией. К тому же некоторые команды недостаточно хорошо раскрыты в официальной документации.

Эмуляторы

В связи с тем, что такие калькуляторы давно не выпускаются, и официальная документация была утеряна в связи с закрытием предприятия, а старые устройства уже, как правило, вышли из строя, для любителей бо́льшее значение приобретают эмуляторы, которые полностью повторяют функциональность ПМК. Это стало возможным после того, как Феликс Лазарев по реальной микросхеме восстановил код ПЗУ и сделал его доступным.

  • Для Андроида есть очень хороший эмулятор: Расширенный эмулятор МК 61/54. И активный форум по нему: Расширенный МК-61.
  • Для разработки можно использовать эмулятор на JavaScript http://mk-61.moy.su/emulator, который автор немного доработал (ограничив только МК-61): Эмулятор МК-61. И составил документацию по нему: Документация по Эмулятору.
